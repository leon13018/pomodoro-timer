## 高頻用戶痛點（來源：Reddit、Medium、App Store 評論、2025-2026 測評文章）

1. **流心狀態中斷**：25 分鐘硬切對需要長時間 ramp-up 的工作（寫程式、寫作）傷害最大
2. **UI 臃腫**：任務管理+番茄合體 App 介面過重，啟動成本高（Focus To-Do、TickTick 都有此問題）
3. **過度遊戲化干擾**：Forest 的種樹機制部分用戶反映分心；功能向訂閱制移動引發「subscription slop」抱怨
4. **整合痛點**：Windows 11 Focus Sessions 只能整合 Microsoft To Do，不支援 Todoist/TickTick；Pomofocus 開發者不回應 issue
5. **Windows 桌面空白**：優質、有設計感的 Windows 原生番茄鐘稀缺。Pomotroid 已加強但仍無任務管理與移動端同步
6. **ADHD 用戶需求**：固定 25/5 分鐘對 ADHD 不友善；需要視覺化倒計時（不只是數字）、靈活暫停、無懲罰機制
7. **Pomofocus 痛點**：開發者長期不回應反饋、Premium 版性價比低、移動端在鎖屏後計時停止
8. **固定休息時長**：Windows 11 Focus Sessions 的 5 分鐘短休無法調整，引發用戶不滿
9. **訂閱疲勞**：Forest 向訂閱制移動後「real tree planting」只限 Plus 用戶，核心賣點被鎖起

## 未滿足需求（市場機會）

- ADHD 友善模式：靈活時長（5-60 分鐘自由設定）、時間可視化（進度條/圓弧清晰顯示）、無強制中斷、暫停不懲罰
- 「靜默桌面陪伴感」：極簡設計、常駐桌面、低侵入性存在感（視窗置頂 + 最小化後仍顯示計時）
- Windows 生態整合：DND 整合、工具列進度顯示（taskbar progress bar）、開機自啟動管理
- 離線優先：無需帳號、無雲端依賴、資料本地存儲
- 視覺差異化：市場上大多數 App 用番茄紅配色，開發者主題（Catppuccin/Nord/Dracula）尚未被充分挖掘
- 工作階段反思：Session app 有「intention setting + reflection」功能，Windows 桌面無對應產品

## 平台空白（2026 更新）

- **Windows 桌面高品質獨立應用**：仍然稀缺。Session/Flow 只有 Mac；Pomotroid 已改善統計但無任務管理與移動同步
- **買斷制 Windows 桌面應用**：幾乎沒有強力競爭者（Focus To-Do $11.99 終生是最接近的）
- **Electron 桌面 + 設計感**：Pomotroid 是主要對手，但其 UI 設計語言偏向極簡扁平，動畫質感有限

**Why:** 識別空白才能找到最省力的差異化路線。

**How to apply:** 優先填補「Windows 高品質桌面」空白，加入視覺化進度（已有 SVG 圓弧），重點補強「Windows 原生整合」（taskbar、DND）與「本地統計資料」。ADHD 友善是中長期加分項。
