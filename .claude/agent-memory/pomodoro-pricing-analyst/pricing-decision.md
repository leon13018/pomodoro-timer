## 最終採用定價

| 項目 | 決定 |
|------|------|
| 模式 | 買斷制（一次付費，終身使用 v1.x） |
| 標準售價 | $12.99 USD |
| 發布優惠 | $9.99（限時 200 名，ProductHunt 發布當天） |
| 試用策略 | 14 天完整功能免費試用，無需信用卡 |
| 付費渠道 | Paddle 或 Lemon Squeezy（MoR，自動處理全球稅務） |
| Microsoft Store | 同步上架，同售價 $12.99 |

**Why:** $12.99 高於最強 Windows 競品 Focus To-Do（$11.99）$1，傳遞「更精緻設計」定位；不選 $9.99 是因為容易被視為廉價 app；不選 $19.99 是因為目前功能尚不支撐該定價。

**How to apply:** 未來討論定價調整時，以 $12.99 為基準錨點。v2.0 大版本可升至 $14.99–$16.99。
